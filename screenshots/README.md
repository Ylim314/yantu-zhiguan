## 重新截图步骤

1. 使用 Node.js 22.x 和 pnpm 10.x 安装项目依赖（`pnpm install`）
2. 启动开发服务器：
   ```bash
   pnpm dev
   ```
3. 在浏览器中打开 http://localhost:5173
4. 使用默认账号登录：用户名 `admin`，密码 `123456`
5. 依次访问上述各页面，截取完整页面截图
6. 将截图按命名规则保存到本目录（`screenshots/`）

## 注意事项

- 当前正式截图使用 Chromium 和 1920×1080 浏览器视口生成
- 如需重新截图，建议使用 Chrome 或 Edge 浏览器，并保持 1920×1080 或 1440×900 视口
- 截图应包含完整的页面内容，包括侧边栏导航
- 对于动态路由页面（任务详情、错题详情），需先点击列表中某一项进入详情页后再截图
